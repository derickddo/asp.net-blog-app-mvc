## Blog App with ASP.NET MVC
This is a simple blog app with ASP.NET MVC. It has a simple UI and a simple database. It has a simple CRUD operation

## Features
- Create a blog
- Read a blog
- Update a blog
- Delete a blog
- Login
- Register
- Logout

## Technologies
- ASP.NET MVC
- Entity Framework
- SQL Server
- HTML
- Bootstrap
